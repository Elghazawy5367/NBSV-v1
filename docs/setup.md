# NBSV GitHub Repo — One-Time Setup Guide
## Browser-Only. No Terminal. No Technical Knowledge Required.
### Complete setup in 30 minutes.

---

## Step 1 — Create Your GitHub Account (5 minutes)

1. Go to `github.com`
2. Click **Sign up**
3. Use your email address
4. Choose a username (something simple — you'll use this in your repo URL)
5. Verify your email

**Free tier is enough for everything in this system.**

---

## Step 2 — Create the Repository (3 minutes)

1. Click the **+** icon (top right) → **New repository**
2. Repository name: `nbsv-engine`
3. Description: `NBSV — Never Before Seen Viral Products — Egyptian Manufacturing System`
4. Visibility: **Private** (your intelligence stays private)
5. Click **Create repository**

Your repo is live at: `github.com/[your-username]/nbsv-engine`

---

## Step 3 — Upload All Files (10 minutes)

1. On your new empty repo page, click **uploading an existing file**
2. On your computer/tablet, find the `nbsv-repo.zip` you downloaded
3. Unzip it — you'll see all the folders
4. Drag the entire folder structure into the GitHub upload window
5. Scroll down → **Commit changes** → click **Commit changes**

All your KB, SOURCES, PROMPTS, CANDIDATES, INTELLIGENCE, SESSIONS, .github folders are now in the repo.

---

## Step 4 — Enable Issues and Issue Templates (2 minutes)

1. Go to repo → **Settings** tab
2. Scroll to **Features** section
3. Make sure **Issues** is checked ✅
4. Issue templates upload automatically with the `.github/ISSUE_TEMPLATE/` folder you uploaded

**Test:** Go to Issues → New Issue. You should see:
- 🎯 NBSV Candidate — New Product
- 🗑️ NBSV Failure Log — Rejected Candidate
- 🏭 Workshop Brief — Production Order

If you see these three — templates are working.

---

## Step 5 — Set Up the Projects Kanban Board (5 minutes)

1. Go to repo → **Projects** tab
2. Click **New project**
3. Select: **Board** (the Kanban view)
4. Name it: `NBSV Pipeline`
5. Create these columns in order (click **+ Add column** for each):

```
🔍 SCANNED
✅ FILTER-PASS
📞 PRE-SELL
🏭 PRODUCTION
🚚 DISTRIBUTED
📊 CLOSED-WON
🗑️ CLOSED-LOST
```

6. Click **Save**

Every candidate Issue you create goes on this board. Drag cards between columns as the product moves through the 7-step cycle.
